Curso de Diseño de Paginas WEB

    Inicio Git Github Html5 CSS3


Objetivo general:

Diseñar una página web para que los estudiantes puedan adentrarce en el mundo del diseño de sitios web, la cual ilustrara información básica que todo diseñador de sitios web debe conocer para empezar a programar y elaborar este tipo de proyectos tecnológicos

Objetivo específico 1:

Conocer las pautas para el desarrollo de un sitio web, tales como tener un objetivo definido, establecer una estructura del sitio, definir el contenido del sitio, organización definida de los contenidos, entre otras recomendaciones básicas para su diseño (Perdomo Vargas, s.f.)

Objetivo específico 2:

Crear un contenido que permita brindar un paso a paso para instalar herramientas como Git y Github su utilidad y manejo dentro del diseño de sitios web

Objetivo específico 3:

Enseñar lenguajes de programación de sitios web tales como HTML5 y CCS3, mediante los cuales se crean el entorno de una página web
